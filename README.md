# jobs-common
Common helpers, configs, and constants used across the Jobs project

## Install

```bash
yarn add @chile-sh/jobs-common
```

## Env variables

```bash
NODE_ENV=development
RMQ_HOST=rabbitmq
RMQ_USER=admin
RMQ_PASS=admin
PG_HOST=postgres
PG_USER=postgres
PG_PASS=postgres
REDIS_HOST=redis
REDIS_PASS=redis
SENTRY_DSN=https://...@sentry.io/1234
```
